Code Reading Club — Music Session

Making Music with Code

Strudel is a live coding environment for making music in the browser. You write code, hit play, and it loops. Change the code while it's running — the music updates on the next cycle. Nothing to install.

open strudel.cc
shortcuts
Ctrl+Enter (⌘+↵ on Mac) play / update
Ctrl+. (⌘+. on Mac) stop
the basics — with strudel.cc open, paste each snippet and listen
The cycle
Everything fits in a repeating loop. More sounds = each gets less time. Fewer = more space.
sound("bd hh sd hh")
// try adding more — it speeds up
Drums
bd bass drum   sd snare   hh hi-hat   oh open hi-hat   cp clap
sound("bd ~ sd ~")
// classic kick-snare
Silence — ~
A tilde is a rest. Holds space without sound. Silence is what makes a beat feel like a beat.
sound("bd sd bd sd")  // no rests
sound("bd ~ sd ~")    // with rests
Repeat — *
Repeat a sound multiple times within one step. hh*4 plays hi-hat four times in that spot.
sound("hh*4")
sound("hh*8")  // even busier
Notes & melody — note()
A melody is a sequence of pitched notes. Use note() with note names + octave numbers. Pair with .sound() to pick an instrument.
note("c3 e3 g3 e3")
  .sound("piano")
Volume — .gain()
1 = full   0.5 = half   0 = silent. Balance layers against each other.
sound("hh*8").gain(0.3)
// hats sit quietly behind kick
Multiple patterns — $:
Start a line with $: to run multiple patterns simultaneously. Each loops independently — this is how you layer drums, bass, and melody.
$: sound("bd ~ sd ~")
$: sound("hh*8").gain(0.3)
$: note("c3 ~ e3 ~ g3 ~ e3 ~").sound("piano")
note names
// white keys — octave 3 and 4 are most useful
c3  d3  e3  f3  g3  a3  b3
c4  d4  e4  f4  g4  a4  b4

// black keys (b = flat, # = sharp)
eb3  ab3  bb3  f#3  c#3

// easy mode — these 5 notes sound good together, pick any of them
c3  eb3  f3  g3  bb3
